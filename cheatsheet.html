üìù Basic Navigation

h j k l Move left/down/up/right

üîô Jump List & Navigation History

Jump Between Locations

Ctrl-o Jump back (older position)
Ctrl-i Jump forward (newer position)
Tab Jump forward (same as Ctrl-i)
:jumps Show jump list
g; Go to older change position
g, Go to newer change position
:changes Show change list

Mark Navigation

ma Set mark 'a' at cursor
'a Jump to line of mark 'a'
`a Jump to position of mark 'a'
'' Jump back to last jump position
`` Jump back to exact position
:marks List all marks

File Navigation History

Ctrl-^ Switch to alternate file
:e # Edit alternate file
g; Previous position in changelist
gi Go to last insert position

Tips:
  • Press :checkhealth to verify your Neovim setup
  • Press <leader>sk to search all keymaps
  • Press <leader>sh to search help docs
  • Most commands can be prefixed with a number for repetition (e.g., 5dd deletes 5 lines)
  • Use . to repeat the last command
  • In fzf/telescope windows: Ctrl-x split, Ctrl-v vsplit, Ctrl-t new tab
  • Ctrl-o and Ctrl-i are your best friends for navigating back and forward through your jump history!
  • Vim automatically adds to the jump list when you: search, use G/gg, jump to marks, or jump to definitions
